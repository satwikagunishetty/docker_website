Description for Docker

To start Docker Services systemctl start docker
To make docker services permanent systemctl enable docker
To show avaliable docker images docker images
To check the running docker containers docker ps
To pull the desired docker image docker pull {imagename:version}
To launch docker container docker run dit --name {osname} {docker img name}
To Stop docker container docker stop {container-name}
To start the stopped docker container docker start {container-name}
To attach volume to docker container docker run -dit --name {os-name} -v {folder-of-baseos}:{folder-of-docker-container} centos
To execute a command inside a docker container docker exec {container-name} {command name}
